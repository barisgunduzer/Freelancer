# Freelancer (Node.js):

Freelancer is a Node.js starter project about portfolio management task
# Using Technologies:
* Express,
* EJS,
* Mongoose,
* Bcrypt,
* Connect-Flash,
* Nodemailer

# Using Methodologies:
* Session Handling,
* User Validation,
* Flash Messages,
* Postman REST Client,

## Usage:

- clone and run `npm i` && `npm start`
